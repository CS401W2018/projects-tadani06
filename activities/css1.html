Explain the difference between inline, embedded, and external CSS?

The difference between inline, embedded, and external CSS is that within inline is focuses on formatting the inner content of block-level elements. While embedded is tag defines a container for an external use, and external CSS is used when you are defining a variety of styles for multiple web pages.
